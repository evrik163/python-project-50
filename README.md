# DESCRIPTION:

**Вычислитель отличий**

Скрипт на вход получает два файла формата YAML или JSON и затем выводит строку, показывающую разницу между ними.

**Запуск справки:**

`gendiff -h`

**Запуск скрипта c настройками по-умолчанию:**

`gendiff <file_path1> <file_path2>`
